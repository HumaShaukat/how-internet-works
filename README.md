# How the Internet Actually Works

An interactive explainer built from scratch — no frameworks, pure HTML/CSS/JS.

## What it covers
- DORA animation — watch DHCP happen step by step
- Live NAT Table — click devices, watch the router's diary fill in real time
- Port Collision simulator — see PAT rename ports when two devices clash
- Full Journey — one Google search from WiFi toggle to page load
- Glossary of all key networking concepts
